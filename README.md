# MajdoorConnect

MajdoorConnect is a hyperlocal platform connecting customers with nearby daily wage workers.

Workers include plumbers, electricians, carpenters, painters and helpers.

Customers can search workers nearby and send job requests.

Workers can accept requests and complete jobs.

Tech Stack

Frontend
React
Vite
Tailwind CSS

Backend
Supabase

Features

OTP login
Worker profiles
Nearby worker search
Booking system
Ratings and reviews
